# GoBlog
一个使用 Go 和 Gin 框架构建的简单博客系统。

## 功能
- 首页模块展示
- 博客文章列表、分页和排序
- 文章详情、评论和回复
- 文章编辑和保存
- 点赞和点踩功能

## 依赖
- Go 1.18+
- GORM (github.com/jinzhu/gorm)
- Gin (github.com/gin-gonic/gin)
- SQLite (gorm.io/driver/sqlite)
